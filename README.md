# Robots First: A City Transformed

An interactive pixel art visualization showing urban transformation from 2025 to 2045 - from car-centric cities to robot-optimized, human-friendly spaces.

## Features

- **Smooth timeline scrubbing** - drag to any year between 2025-2045
- **Auto-play mode** - press SPACE to watch the transformation
- **Four detailed zones** - click tabs to switch views
- **Pixel art aesthetic** - nostalgic visual style
- **No dependencies** - single HTML file, runs anywhere

## Usage

Just open `index.html` in any modern browser. No build process, no dependencies.

Or visit the live demo link above.

## Controls

- **Drag slider** - Scrub through time
- **SPACE** - Toggle auto-play
- **Click tabs** - Switch between zones

## Tech

Pure HTML5 Canvas + vanilla JavaScript. No frameworks, no build tools, no external dependencies.
